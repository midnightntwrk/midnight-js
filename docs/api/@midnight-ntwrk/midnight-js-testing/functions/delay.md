# Function: delay()

> **delay**(`ms`): `Promise`\<`unknown`\>

Creates a Promise that resolves after a specified delay.

## Parameters

### ms

`number`

The delay duration in milliseconds.

## Returns

`Promise`\<`unknown`\>

A Promise that resolves after the specified delay.

## Example

```ts
// Wait for 1 second
await delay(1000);
```
